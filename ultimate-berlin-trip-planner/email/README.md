# Ultimate Berlin Trip Planner Email Funnel

Source copy for Wix Triggered Emails used by the booking-aware trip planner
sequence.

Use the sales-path files while the lead has not booked. Use the `booked-*` files
after the backend marks a matching lead as booked.

Variables expected by Velo:

- `${arrivalDate}`
- `${tripLength}`
- `${planTitle}`
- `${recommendedTourDay}`
- `${ticket}`
- `${weatherTitle}`
- `${arrivalTime}`
- `${arrivalPoint}`
- `${stayArea}`
- `${groupType}`
- `${interests}`
- `${budgetStyle}`
- `${mustHandle}`
- `${pace}`
- `${tourIntent}`
- `${bookingStatus}`
- `${tourDate}`
- `${bookingUrl}`
- `${meetingPointUrl}`
- `${firstDayPlannerUrl}`
- `${ticketCalculatorUrl}`
- `${whatsOpenUrl}`
- `${dailyBudgetUrl}`

Tone: practical, calm, local-guide voice. Public links should use
`berlinwalk.com` and `@berlinwalkingtour` only.
