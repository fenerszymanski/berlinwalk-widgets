BerlinWalk launch kit

Ultimate Berlin Trip Planner Triggered Emails

Use Wix Developer Tools -> Triggered Emails, not an automation workflow. Create the five planner templates, paste each subject, preheader, and HTML body, then paste the saved editor URL or message ID back into the matching input. Do not create booked-path planner templates; the existing booking email sequence already handles booked guests.

Message ID JSON

After all five templates are saved in Wix, copy or download this JSON as message-ids.local.json. If it lands in Downloads, run import-message-ids-from-downloads.mjs, then run the apply helper from the repo root.
