# Stolpersteine Berlin — Commons image candidates

Scouted 2026-07-20 via the Wikimedia Commons API. All licenses verified per file
through `prop=imageinfo&iiprop=extmetadata`. Buckets: (1) cover close-up,
(2) group/row at doorway, (3) Demnig/laying, (4) polished-vs-weathered /
cleaning / flowers, (5) wider street scene, (6) specific locations
(Grosse Hamburger Str, Oranienstrasse Kreuzberg).

All Commons URLs below have parentheses percent-encoded (blog markdown parser
truncates raw parens).
