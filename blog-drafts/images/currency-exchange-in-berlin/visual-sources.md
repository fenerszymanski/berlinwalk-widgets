# Currency Exchange in Berlin — Visual Sources

Post: `currency-exchange-in-berlin`
Sourced 2026-07-11. All images are freely licensed (Wikimedia Commons). No paid
API or AI generation used. Germany has Freedom of Panorama, so photos of public
storefronts/machines are fine to use.

## Final set (optimized/)

1. **01-currency-exchange-berlin-euro-cash-cover.jpg** — FEATURED / cover
   - Subject: Close-up of euro banknotes (€50, €20). Clean, colourful, instantly
     reads as "euro cash" on a listing card and OG image. Landscape 3:2.
   - Source: File:EURO Cash Money Banknote (49643807026).jpg
   - URL: https://commons.wikimedia.org/wiki/File:EURO_Cash_Money_Banknote_(49643807026).jpg
   - Author: Markus Spiske · License: CC BY 2.0 (attribution required)

2. **02-currency-exchange-berlin-bank-atm-sparkasse.jpg** — bank-ATM section
   - Subject: Sparkasse Geldautomat (bank ATM) with lit orange columns at
     Möckernbrücke, Berlin. Distinctive, Berlin-specific, on-message ("use a
     real bank ATM"). Portrait.
   - Source: File:Sparkassen Geldautomat Moeckernbruecke 26.10.2014 10-50-18.JPG
   - URL: https://commons.wikimedia.org/wiki/File:Sparkassen_Geldautomat_Moeckernbruecke_26.10.2014_10-50-18.JPG
   - Author: Dirk Ingo Franke · License: CC BY-SA 4.0 (attribution required)

3. **03-currency-exchange-berlin-euronet-travel-money-atm.jpg** — the "avoid" section
   - Subject: Euronet "Reisegeld-Automat / Travel Money ATM" with clear Euronet
     branding. The exact standalone machine tourists should avoid. Portrait.
   - Source: File:Euronet Reisegeld-Automat (Travel Money ATM) in Munich.jpg
   - URL: https://commons.wikimedia.org/wiki/File:Euronet_Reisegeld-Automat_(Travel_Money_ATM)_in_Munich.jpg
   - Author: Wilfredor · License: CC0 (public domain, no attribution required)

4. **04-currency-exchange-berlin-spaetkauf-cash-only.jpg** — cash-in-Berlin section
   - Subject: Colourful Spätkauf (Späti) storefront in Berlin-Kreuzberg. Anchors
     the "you still need some cash" point (Spätis, markets, small bars). Landscape.
   - Source: File:Spätkauf in Berlin-Kreuzberg.jpg
   - URL: https://commons.wikimedia.org/wiki/File:Sp%C3%A4tkauf_in_Berlin-Kreuzberg.jpg
   - Author: Nicolas Bouliane · License: CC BY-SA 4.0 (attribution required)
