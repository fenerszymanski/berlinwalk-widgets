# Visual sources — Hitler's bunker (Führerbunker) site

Daily-blog package `hitlers-bunker-berlin`. All images sourced from Wikimedia
Commons, downloaded 2026-07-13. No paid/AI image generation used. Raw originals
in `raw/`, web-optimized copies in `optimized/`.

Sensitive-topic rule applied: only honest documentary photographs (the site
today, the information board, the ruined Reich Chancellery in 1945, the
neighbouring Holocaust Memorial). No Hitler portraits, no glorifying or
shrine-style imagery, no fabricated scenes.
