## Rejected / Quality Notes

- Generic hospital PDFs, archive scans and old medical consultation images were rejected because they are not relevant to a modern Berlin tourist medical-help guide.
- A Berlin Hauptbahnhof context image was rejected from the final set because it was too generic for a medical-help article.
- The Charite image is kept inline only; it is too distant for featured-image use.
- No protest, accident-scene, distressing injury, blood, or event-specific emergency images were used.
- Public post includes an Image Credits block because CC BY / CC BY-SA items require attribution and the Wikimedia set is more transparent with credits.
