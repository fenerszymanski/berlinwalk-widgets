## Contact Sheet / Quality Gate

- Candidate sheet: `contact-sheets/candidate-contact-sheet.jpg`
- Cover crop check: `contact-sheets/cover-crop-check.jpg`
- Final sheet: `contact-sheets/final-contact-sheet.jpg`

Quality decision:

- Cover passed because it shows the core site elements in one glance and stays legible in a 16:9 crop.
- Documentation centre image kept inline only because the architecture is clean but visually sterile.
- Repetitive wide-angle candidates were rejected or limited to one support image.
- No dull generic street scene, protest/event-specific photo, misleading archival-style image or fabricated visual was used.
