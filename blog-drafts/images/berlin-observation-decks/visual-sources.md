# berlin-observation-decks — visual sources

All images are Wikimedia Commons, license-checked via the Commons API
(`imageinfo` + `extmetadata`) on 2026-07-12. Optimized to 1600px longest edge,
JPEG quality ~82, saved under `optimized/`. A reader-facing Image Credits block
is included in the published body per the CC attribution requirement.

## Facts checked 2026-07-12 (sources in local notes)
- TV Tower deck 203 m, adult from ~€24.50 online, dynamic pricing, 40s lift.
- Panoramapunkt / Kollhoff Tower ~100 m, €9 (reduced €7), fastest lift in
  Europe ~20s, open-air terrace + Potsdamer Platz open-air exhibition.
- Park Inn Panorama Terrasse ~120 m (39th floor), ~€6, open roof, walk-up.
- Reichstag dome free, advance ID registration required, open ~8:00–24:00.
- Victory Column ~51 m platform, 285 steps, ~€4 cash, no lift.
