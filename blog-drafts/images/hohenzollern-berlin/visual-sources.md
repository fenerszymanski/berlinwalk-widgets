# Hohenzollern Berlin Visual Sources

Run date: 2026-06-29

## Quality Gate

Final set passed the visual quality gate in a local contact sheet (`contact-sheet-final.jpg`). The cover is topic-specific and readable at card crop size; support images each carry a concrete article section. Rejected direction: generic Berlin street views, weak palace facade crops, crowded event/protest scenes, dull documentation images, and images that were legally usable but not strong enough editorially.

No paid image API was used. No article image uses generated archival evidence.
