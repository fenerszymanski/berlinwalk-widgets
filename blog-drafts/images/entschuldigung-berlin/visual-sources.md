## Rejected

- `Wiki Loves Cocktails 2017, Berlin, 20170716 (DSC06538).jpg` (Velvet Bar,
  Neukölln) — too dark, faces unreadable, reads as a photography-meetup event
  shot rather than an everyday café/bar moment. Did not clear the visual
  quality bar; no bar/café inline image was used in the final set (5 images
  without it already cleared the 4-image baseline with good topic coverage).
- Several older black-and-white U-Bahn/Bundesarchiv images were considered
  for historical texture but dropped — the post is about a present-day
  practical word, not transit history, so a period photo would have been
  misleading context.
