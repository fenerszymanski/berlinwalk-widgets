# KaDeWe Berlin — Visual Sources

All images sourced from Wikimedia Commons. No paid/AI image generation used.
Optimized to progressive JPEG, ~1500px wide (cover 1600px). Raw originals in
`raw/`, delivery files in `optimized/`.

Reader-facing Image Credits block required for the five CC BY / CC BY-SA
images (cover + food hall + patisserie + winter garden + Wittenbergplatz). The
1907 entrance-hall photo is public domain (no required credit, credited for
context only).

## Selected

1. **01-kadewe-berlin-cover.jpg** (cover / featured)

   - https://commons.wikimedia.org/wiki/File:KaDeWe_von_Nordosten_(2008).jpg

   - Why: clean, complete facade of the whole building with the red roofline
     and blue sky, instantly recognizable at listing-card size. Best editorial
     cover of the set.

   - Source: `File:KaDeWe Deli.JPG`
