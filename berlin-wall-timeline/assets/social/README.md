# Timeline social cover

`berlin-wall-timeline-1200x630.jpg` is a 1200×630 crop of the historical
photograph **Crane removed part of Wall Brandenburg Gate.jpg** by SSGT F. Lee
Corkran, sourced from Wikimedia Commons:

https://commons.wikimedia.org/wiki/File:Crane_removed_part_of_Wall_Brandenburg_Gate.jpg

The source is a U.S. federal government photograph marked public domain on the
Commons file page. It shows the Brandenburg Gate opening beside the Wall on 21
December 1989.
