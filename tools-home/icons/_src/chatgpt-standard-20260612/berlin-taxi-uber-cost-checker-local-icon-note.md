# Berlin Taxi or Uber Cost Checker Icon Note

Tool slug: `berlin-taxi-uber-cost-checker`

Created on 2026-06-14 during the daily blog draft automation.

The project standard prefers ChatGPT-browser-generated glossy 3D icons for new BerlinTools cards. This automation did **not** use paid image/video generation APIs. To keep the tool card from shipping with a fallback letter or unrelated image, Codex created a local deterministic RGBA PNG icon with Pillow using the existing BerlinWalk glossy-icon constraints:

- square green app tile
- yellow medallion
- simple taxi object
- no text labels
- 512px and 160px PNG outputs

Canonical files:

- `berlinwalk-widgets/tools-home/icons/berlin-taxi-uber-cost-checker.png`
- `berlinwalk-widgets/tools-home/icons/berlin-taxi-uber-cost-checker-160.png`
