# Berlin Trip Planner Homepage Teaser

Compact homepage Custom Element for the standalone `/berlin-trip-planner` page.

## Wix install

```html
<bw-trip-planner-home></bw-trip-planner-home>
<script src="https://fenerszymanski.github.io/berlinwalk-widgets/trip-planner-home/trip-planner-home-element.js?v=20260606"></script>
```

Use as a normal homepage section, ideally near the planning/tools area or below
the main hero once the page is ready to promote.
