# First-Day Planner Lead Funnel Emails

Source copy for the four Wix Triggered Emails used by the V3 lead funnel.

Use these files as the editor copy deck, then copy the resulting Triggered Email message IDs into `../velo/firstDayPlannerFunnel.js`.

Variables expected by Velo:

- `${arrivalDate}`
- `${planTitle}`
- `${ticket}`
- `${tourFit}`
- `${weatherTitle}`
- `${startPoint}`
- `${bookingUrl}`
- `${meetingPointUrl}`
- `${ticketCalculatorUrl}`
- `${whatsOpenUrl}`
- `${berlinThreeDaysUrl}`

Tone: practical, calm, local-guide voice. Public links should use `berlinwalk.com` and `@berlinwalkingtour` only.
