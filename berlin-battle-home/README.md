# BerlinWalk Games Home Section

Homepage Custom Element teaser for the BerlinWalk Games hub and the three
current games: Berlin Battle, Berghain Bouncer, and Berlin Smile Challenge.

The custom element name stays `<bw-berlin-battle-home>` for live Wix embed
compatibility.

Use on Wix:

```html
<bw-berlin-battle-home></bw-berlin-battle-home>
<script src="https://fenerszymanski.github.io/berlinwalk-widgets/berlin-battle-home/berlin-battle-home-element.js?v=games-home-three-games-20260624"></script>
```

Recommended placement: after the planning/tools area and before the editorial
blog teaser. It should feel like a playful discovery break, not the first
conversion hero.
