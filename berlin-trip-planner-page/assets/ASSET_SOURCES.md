## Prompt

Create one polished website asset sheet for BerlinWalk's Berlin Trip Planner
landing page.

Output: one single PNG image, 16:9, arranged as a clean grid on a transparent or
very light cream background.

Top row: four equal square app-style visual icons, no text/letters/numbers:
1. 1-7 day trip plan: calendar pages + small route path.
2. Weather aware: cloud/sun/rain with a small Berlin landmark silhouette.
3. Map ready: folded map + location pin + walking route line.
4. Local guide: friendly guide notebook + speech/story cue, no person face.

Bottom row: three wide process illustrations, no text/letters/numbers:
1. Arrival reality: suitcase, airport/train arrival, city ticket, simple route
   into Berlin.
2. Route respects energy: calm Berlin neighborhood street, cafe pause, route
   staying in one area.
3. BerlinWalk at the right time: World Clock/Alexanderplatz meeting point,
   walking group cue, route context.

Style: premium editorial illustration, soft oil-paint texture mixed with modern
UI icon clarity, rounded shapes, high contrast, warm but restrained. Use
BerlinWalk palette: dark green #1B5E20, yellow #FFE600, lime #7CB342, cream
#FAFAF5, small blue accent only if needed. No visible words, no labels, no
watermarks, no logos, no text at all. Keep each cell visually separated so it
   can be cropped into individual assets.
